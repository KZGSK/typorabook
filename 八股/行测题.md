# 行测题

##数资

###**现期、基期**

现期 A ；基期 B ；增长率r；增长量R

B = A/(1+r)

B = A-R

一般问现在的数据 A或者过去的数据B，给出增长率进行计算。

###**增长率**

**一般增长率**

问增长了百分之多少

r = (A-B)/B = R/B

**间隔增长量**

知道2013年、2012年增长率，问2013年相对2011年增长率

r = r1 + r2 + r1 ×r2

**混合增长率**

混合后的增长率r介于r1、r2之间，并且偏向现期数量大的一方。 

###增长量

R = A-B

R = A×r/(1+r)

### 平均数

问C每D，如多少万吨/千公顷，吨/亩

对于现期直接÷：C/D

基期平均值先÷现期，再看后面增长率，大于1就大，小于1就比现期小

(C/D)*((1+rc)/(1+rd))

###平均数增长率

n = (rc-rd)/(1+rd)

c是分子增长率。d是分母增长率
